# Provisionamento Financeiro – Excel VBA

Projeto em Excel VBA que permite ao usuário selecionar um arquivo externo,
importar automaticamente os dados para o Excel e gerar um dashboard
financeiro com indicadores e gráficos atualizados.

O objetivo é facilitar o processo de provisionamento financeiro,
eliminando tarefas manuais e reduzindo erros.

## Funcionalidades
- Seleção de arquivo pelo usuário (FileDialog)
- Importação automática de dados
- Tratamento e padronização das informações
- Atualização dinâmica das tabelas
- Dashboard financeiro com gráficos
- Botão de atualização automática

## Dashboard Financeiro

O dashboard permite análise dinâmica do provisionamento financeiro,
com filtros por mês, centro de custos e ordem interna.

Principais indicadores:
- Total provisionado
- Média de gastos por fornecedor
- Distribuição de gastos por conta contábil
- Ranking dos 5 maiores gastos
